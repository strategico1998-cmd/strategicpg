# STRATEGIC - Performance Architecture 🛡️

**Chief Conversion Officer 2026** - Ingeniería de Landing Pages de alto rendimiento.

## 🚀 Descripción
Este repositorio contiene la arquitectura técnica de la landing page de **STRATEGIC**. Enfocada en la ingeniería de persuasión, Paid Media y optimización de conversión (CRO) para el mercado 2026.

### 🛠️ Nuevas Funcionalidades
- **Calculadoras de Inversión Duales:** Estimación en tiempo real para campañas de *Leads/Citas* y *E-commerce/Ventas* con rangos desde $3M.
- **Planes de Servicio Escalables:** Propulsor Orbital, Órbita de Crecimiento y Expansión Galáctica.
- **Acceso a Recursos Elite:** Sistema de descarga de herramientas técnicas directamente integrado.
- **Tracking Avanzado:** Integración nativa con Microsoft Clarity y GTM.

## 💻 Desarrollo Local
Para visualizar los cambios localmente:
1. Clona el repositorio.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta `npx serve .` o abre `index.html` directamente en tu navegador.

## 📤 Despliegue
El proyecto está configurado para ser desplegado automáticamente en **Vercel** o **GitHub Pages**.
